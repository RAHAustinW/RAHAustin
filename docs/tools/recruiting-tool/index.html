    How this was built

    Demand is measured from delivered & scheduled care logs (27,832 shifts, Jan–Jun 2026) — actual visits, not open shifts. Hours = scheduled clock-out minus clock-in, capped at 24h. Supply is the count of active caregivers (310) by home ZIP. "Local supply gap" (strain) = demand hours ÷ active caregivers living in that ZIP. ZIPs geocoded to centroids; map tiles © OpenStreetMap.

    Read with these caveats

    Care logs show demand that was covered — true unmet need (declined referrals, open shifts) isn't in this export, so gaps are likely understated. The caregiver roster has no availability data, so "who can work overnights/weekends" is inferred from who actually did. To sharpen this: pull an Open/Unfilled Shifts export and a caregiver availability export next.
