Standard schema
ENTITY_ PAYER_ LASTNAME_ FIRSTNAME_ DOCTYPE_ YYYY-MM-DD_ SOURCE _ EXT-ID
PAYER applies to clients only. EXT-ID is optional — use only when two people share the same full name.
